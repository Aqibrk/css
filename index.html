CSS Learning Guide

Master CSS concepts with practical examples

Created by Sir Aqib Khan Life Editor
Basic Intermediate Advanced

1. CSS Introduction

An overview of CSS and its purpose in styling web pages.

body {
background-color: lightblue;
}

2. CSS Syntax

The structure of CSS, consisting of selectors and declarations.

h1 {
color: blue;
}

3. CSS Selectors

Patterns used to select elements for styling.

p {
color: green;
}

4. CSS Colors

Define colors using names, HEX, RGB, HSL, etc.

h1 {
color: rgb(255, 0, 0);
}

5. CSS Backgrounds

Set the background color, image, or gradient of elements.

body {
background-image: url('background.jpg');
}

6. CSS Borders

Add borders around elements.

div {
 border: 2px solid black;
}

7. CSS Margins

Define space around elements.

div {
margin: 20px;
}

8. CSS Padding

Define space inside elements between content and borders.

div {
padding: 15px;
}

9. CSS Box Model

Model describing the layout of elements, including margins, borders, padding, and content.

div {
margin: 10px;
padding: 20px;
border: 5px solid black;
}

10. CSS Text

Style text with properties like color, alignment, and decoration.

p {
text-align: center;
color: darkblue;
}

11. CSS Fonts

Specify font families, sizes, and weights.

h1 {
font-family: Arial, sans-serif;
font-size: 36px;
}

12. CSS Links

Style hyperlinks in their various states.

a {
color: blue;
text-decoration: none;
    }
a:hover {
    color: red;
}

13. CSS Lists

Style ordered and unordered lists.

ul {
list-style-type: square;
}

14. CSS Tables

Style HTML tables, including borders and spacing.

table {
border-collapse: collapse;
 }
td {
border: 1px solid black;
}

15. CSS Positioning

Position elements using properties like static, relative, absolute, and fixed.

div {
position: relative;
top: 10px;
left: 20px;
}

16. CSS Display

Control the display type of elements (block, inline, none, etc.).

p {
 display: block;
 }

17. CSS Visibility

Control the visibility of elements.

div {
 visibility: hidden;
}

18. CSS Flexbox

A layout module for aligning and distributing elements in a container.

.container {
display: flex;
justify-content: space-between;
}

19. CSS Ordered List

Customize the appearance of ordered lists.

ol {
list-style-type: upper-roman;
}

20. CSS Float

Float elements to the left or right.

img {
float: left;
}

21. CSS Overflow

Handle content that exceeds an element's box.

div {
overflow: hidden;
}

22. CSS Z-Index

Set the stack order of elements.

div {
z-index: 10;
}

23. CSS Units

Specify sizes using units like px, em, rem, %, vw, and vh.

div {
width: 50%;
height: 200px;
}

24. CSS Media Queries

Create responsive designs that adapt to different screen sizes.

@media screen and (max-width: 600px) {
body {
 background-color: lightgreen;
}
}

25. CSS Animations

Create animations using @keyframes and animation properties.

@keyframes slide {
from { left: 0; }
to { left: 100px; }
}
div {
position: absolute;
animation: slide 2s;
}

26. CSS Transitions

Add smooth transitions between states of an element.

button {
background-color: blue;
transition: background-color 0.3s ease;
}
button:hover {
background-color: green;
}

27. CSS Transform

Apply transformations like rotate, scale, and translate to elements.

div {
transform: rotate(45deg);
}

28. CSS Variables

Define reusable values for CSS properties.

:root {
--main-color: #3498db;
}
div {
color: var(--main-color);
}

29. CSS Pseudo-classes

Style elements based on their state or position.

a:hover {
color: red;
}

30. CSS Pseudo-elements

Style specific parts of an element, like ::before and ::after.

p::before {
content: "Note: ";
font-weight: bold;
        }

31. CSS Opacity

Control the transparency of elements.

div {
opacity: 0.5;
}

32. CSS Shadows

Add box-shadow or text-shadow to elements.

div {
box-shadow:
5px 5px 10px rgba(0, 0, 0, 0.5);
}

33. CSS Gradients

Create gradient backgrounds using linear or radial gradients.

body {
background: linear-gradient
(to right, red, yellow);
}

34. CSS Clip Path

Clip elements to a specific shape.

div {
clip-path: circle(50%);
}

35. CSS Filters

Apply visual effects like blur and brightness to elements.

img {
filter: blur(5px);
}

36. CSS Scrollbars

Style scrollbars for custom designs.

img {
object-fit: cover;
}

37. CSS Counters

Create custom counters for lists or other elements.

ol {
counter-reset: item;
}
li {
counter-increment: item;
}
li::before {
content: counter(item) ". ";
}

38. CSS Writing Modes

Control text direction and orientation.

div {
writing-mode: vertical-rl;
}

39. CSS Responsive Design

Techniques for making web pages adaptable to different devices.

@media screen and (max-width: 600px) {
    body {
font-size: 14px;
    }
}

40. CSS Frameworks

Predefined CSS libraries like Bootstrap or Tailwind CSS.

@import url
('https://cdnjs.cloudflare.com

ajax/libs/bootstrap/
4.5.0/css/bootstrap.min.css');

41. CSS Best Practices

Guidelines for writing clean and maintainable CSS.

/* Use meaningful class names */
.button {
background-color: blue;
}

42. CSS Debugging

Techniques for troubleshooting CSS issues.

div {
border: 1px solid red;
}

43. CSS Important

Override all other styling using the !important keyword.

p {
color: green !important;
}

44. CSS Shorthands

Combine multiple CSS properties into a single declaration.

div {
margin: 10px 20px 30px 40px;
}

45. CSS border

Add borders around elements

div {
border: 2px solid black;
background-color: lightgray;
                }
